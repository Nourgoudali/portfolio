## ğŸ“– About The Project

This project is a modern, responsive personal portfolio website designed to serve as a digital resume and showcase of work. It is built as a Single Page Application (SPA) using vanilla HTML, CSS, and JavaScript, ensuring fast load times and a smooth user experience without the overhead of heavy frameworks.

### Key Features

*   **Responsive Design:** Fully responsive layout that adapts to all screen sizes (Desktop, Tablet, Mobile).
*   **Mobile Sidebar:** Collapsible sidebar for seamless mobile navigation.
*   **Categories & Filtering:** Dynamic project filtering allows visitors to sort portfolio items by category (Web Design, Applications, Web Development).
*   **Testimonials Modal:** Interactive modal for viewing client testimonials with details.
*   **Contact Form Validation:** Real-time form validation to ensure data integrity before submission.
*   **SPA Navigation:** Smooth page transitions without reloading the browser.
*   **Modern Aesthetics:** Uses distinct deep dark color schemes (`eerie-black`, `smoky-black`, `ONYX`) with vibrant accents.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   A modern web browser (Chrome, Firefox, Safari, Edge).
*   A text editor (VS Code, Sublime Text, etc.) if you wish to edit the code.

### Installation

1.  **Clone the repository**
    ```sh
    git clone https://github.com/Nourgoudali/portfolio.git
    ```
2.  **Navigate to the project directory**
    ```sh
    cd portfolio
    ```
3.  **Open `index.html`**
    *   Simply double-click `index.html` to open it in your default browser.
    *   OR serve it using a local server extension (like Live Server in VS Code) for the best experience.

## âœ¨ Usage

1.  **Sidebar**: On mobile, use the top-left button to toggle your contact info.
2.  **Navigation**: Use the bottom (or side) navbar to switch between **About**, **Resume**, **Portfolio**, and **Contact** sections.
3.  **Portfolio Filter**: Click on category buttons (or use the dropdown) in the Portfolio section to filter projects.
4.  **Testimonials**: Click on any testimonial item to open a detailed modal view.
